Вычислительный экперимент к статье из журнала Квант №10 2021г. "Нетранзитивные кости".
В статье даются оценки максимума для суммы вероятностей выигрышей в случае трёх костей с достаточно сложным доказательством.
В тоже время оценки можно получить прямым перебором. Количество комбинаций для 3-х костей: 18!/(6!6!6!) = 17153136, что легко доступно для перебора.
Т.о. есть возможность проверить как саму оценку, так и её достижимость.
Заодно можно посчитать и количество нетранзистивных костей.
В конце статьи делается замечание о возможности расширить результат для N >= 3 костей. 
Здесь перебор уже, конечно, не поможет.

P.S. Оценка верна и достижима. Нетранзистивных костей не так и много: 32115 из 17153136.
